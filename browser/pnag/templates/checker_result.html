Predicted melting temperature (Tm, in °C) for ASOs

The predicted Tm of ASOs are shown in the barplot below. For Tm prediction, we used the R package rmelting (v1.10.0) which is an interface to the MELTING (v5) program. We calculate the melting temperature for RNA-RNA duplexes, because there are no available algorithms for the Tm of PNA-mRNA duplexes. As parameters we used the nucleic acid concentration of 8 μM and a Na concentration of 0.1M. The Tm values are useful for relative comparison between PNAs. However, the absolute Tm is only an approximation and should be validated experimentally. In general, PNAs with a very low Tm should be avoided to enable PNA-mRNA binding.
